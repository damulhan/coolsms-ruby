### COOLSMS SDK for Ruby 

config.json 파일을 아래와 같이 설정 후 examples 아래 예제 코드를 실행해 보세요.
```
{
  "api_key": "[INPUT YOUR API KEY]",
  "api_secret": "[INPUT YOUR API SECRET]",
  "protocol": "https",
  "domain": "api.coolsms.co.kr",
  "prefix": ""
}
```

### 설치 

```ruby
gem 'coolsms', github: 'damulhan/coolsms-ruby'
```

### 링크
